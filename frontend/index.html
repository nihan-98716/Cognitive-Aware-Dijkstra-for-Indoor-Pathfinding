RESEARCH BASIS

                    

                        
                    

                

            


            
            

                

                    
Classical Dijkstra

                    
Finds the shortest path based purely on physical distance. Optimal for minimizing travel distance but ignores cognitive complexity.

                

                

                    
Cognitive Dijkstra

                    
Modified Dijkstra using weighted cognitive factors. Prioritizes paths that are easier to navigate mentally, even if longer.

                

                

                    
A* Algorithm

                    
Uses heuristic (Euclidean distance) to guide search toward goal. More efficient than Dijkstra for single-target pathfinding.

                

                

                    
Bellman-Ford

                    
Handles negative edge weights and detects negative cycles. Slower but more versatile than Dijkstra.
